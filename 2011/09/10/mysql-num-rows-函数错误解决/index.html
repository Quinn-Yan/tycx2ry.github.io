mysql_num_rows() 函数错误解决

Veröffentlicht am 2011-09-10 | in PHP|MYSQL学习

     今天下午写代码时候出现了这种错误Warning: mysql_num_rows() expects parameter 1 to be resource, boolean given in??????

      我起初是以为是不是那个变量打错了，但是没有！再后来我就百度！没有找到答案！然后谷歌，全是英文，但是还是基本看懂了意思！

      原来错误处在sql语句中的一个空格问题！

      看代码错误的：

 $commsql= “select name from comments where blog_id=” . $row[’id’] . “order by dateposted;”;

     这个php代码提交到服务器上就是 select name from comments where blog_id=1 order bydateposted;看见没有order by后面应该有个空格的！

      正确代码是：

 $commsql= “select name from comments where blog_id=” . $row[’id’] . “ order by dateposted;”;

        在order前加一个空格！结果就会这样：

     select name from comments where blog_id=1 order by dateposted;
